# Analiza Agentie Imobiliara

## Descriere
Acest proiect are ca scop analiza datelor unei agentii imobiliare, pentru a intelege distributia preturilor si caracteristicile proprietatilor.
Aplicatia este realizata folosind Streamlit si permite vizualizarea rapida a datelor.

## Tehnologii utilizate
- Python
- Streamlit
- Pandas
- Matplotlib

---

## Functionalitati
- Incarcarea si afisarea datelor din CSV
- Vizualizarea primelor inregistrari
- Statistici descriptive ale datelor
